Design Course Projects

ENGINEER 1P13 @ McMaster University

Project 1 — International Airport

This project focused on the design and simulation of multiple subsystems within an international airport environment. A passenger ticket check-in system was developed using Python, a luggage handling mechanism was designed using Autodesk Inventor, and Q-arm code was implemented to control the movement of luggage across the designed mechanism, simulating automated baggage sorting and system coordination.

Project 2 — Wastewater Filtration

This project focused on the design and analysis of a wastewater filtration system with an emphasis on material selection and sustainability. Granta EduPack was used to evaluate candidate filtration materials based on mechanical and physical properties, perform eco-audits to assess environmental impact, and compare options using Ashby charts. Technical calculations, including porosity analysis, were conducted to assess filtration effectiveness and support design decisions.

Project 3 — Multi-sensory Cooking Thermometer

This project focused on the design and implementation of a multi-sensory cooking thermometer using an Arduino embedded system in C/C++. Temperature sensors were integrated with audio (speaker) and visual feedback (LED) components to provide live alerts, and code was written to process sensor data and trigger outputs, improving accessibility and safety during cooking.
