DC Fast EV Charger Interface

High Power Electronics Lab (HiPEL) @ McMaster University

Introduction to the Project

This project focuses on the design of a multi-screen user interface for a DC fast electric vehicle charger with vehicle-to-vehicle (V2V) capability. The system was programmed in C/C++ using the Arduino IDE and communicates with charging hardware through Modbus, CAN, and RS485 shield protocols.

The interface guides users through a charging session, beginning with system initialization and communication verification with the charging module, followed by configuration of charging parameters. Multiple screens allow users to select charging modes. Live charging data is continuously acquired through the CAN shield and RS485 modules and displayed. A summary screen is provided for each completed charging session.

Components Used

  • Arduino GIGA R1 WiFi
  • Arduino GIGA Display Shield
  • CAN-BUS Shield V2.0
  • Arduino RS485 Shield

Skills Acquired

  • C/C++ embedded systems programming in the Arduino IDE
  • Modbus, CAN, and RS485 communication protocol implementation
  • Touchscreen UI development
  • System communication verification and error handling
  • Hardware wiring of Arduino with CAN and RS485 shields
